# 常用工具函数封装
```
// 加载 axtools 整体内容(min)
import axtools from 'axtools';

// 加载 axtools 的指定类型的全部内容(min)

// 加载 axtools 的指定方法
import {deepCopyObj} from 'axtools/utils/object';

```
